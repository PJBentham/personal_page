My Sign Off - Github - github.com/PJBentham/MySignOff_PHP


Core skills required: HTML / CSS / Javascript / Jquery / PHP / MySQL


Where I’m currently employed as an Account Manager (TSF Retail Solutions), sign off documents are often required – the logistics of getting hard copies of the documents back to the office in a timely manner has proven troublesome. To combat this I have designed a site that will allow online sign off forms to be completed on a mobile device with a real signature. This form and the data along with it, is made into a PDF which is uploaded to a database for master users (managers) to see and download as and when required.

Example returned sign off form

Librarys used & other info:


Javascript:

Signature Pad - https://github.com/thomasjbradley/signature-pad
A jQuery plugin for assisting in the creation of an HTML5 canvas based signature pad. Records the drawn signature in JSON for later regeneration.

Sort Table - http://www.kryogenix.org/code/browser/sorttable/
Allows the sorting of a table of data without having to resubmit the page

PHP:

FPDF - http://www.fpdf.org/
Creates PDF files via PHP, so I could generate and store PDFs from the submitted forms

PHPMailer - https://github.com/PHPMailer/PHPMailer
Allows the PDF'd sign off forms to be emailed to the relevant manager once submitted by site fitter

PDO - http://php.net/manual/en/intro.pdo.php
PHP class to interact with the MySQL database

Secure Login - http://www.wikihow.com/Create-a-Secure-Login-Script-in-PHP-and-MySQL
A secure method for storing user info and allowing both the managers and fitters to sign in and out
