Halloween Pi Project - Youtube


Core skills required: Python


A fun halloween project using the raspberry pi. I set up a button through the IO pins on the pi, when the button was pushed the pi would play some spooky sounds and take a photograph. The python program then overlayed a transparent picture of the Grim Reaper on the photograph.
