Website - Mysignoff - March 2014

Website: MySignOff

PHP/HTML Form Project

Core skills required:
HTML / CSS / Javascript / Jquery / PHP / MySQL Where I’m currently employed as an Account Manager (TSF Retail Solutions), sign off documents are often required – the logistics of getting hard copies of the documents back to the office in a timely manner has proven troublesome. To combat this I have designed a site that will allow online sign off forms to be completed on a mobile device with a real signature. This form & the data along with it is made into a PDF which is uploaded to a database for master users to see and download as and when required.

About Me

I'm a retail account manager by day and a freelance developer by night. I've been learning to program over the past three / four years, I REALLY enjoy it. I've worked at my current job for the past nine years and would like to think I'm fairly competent in my field.

I first started programming pretty much when Codacademy came online and made it so easy to learn. I began with the Python course and eventually looked at the HTML, CSS and Jquery courses. I'm far more interested in Python though as I like to write programs that do things rather than pretty websites, I am far from a good designer!

I'm a big fan of the Raspberry Pi and have completed a good few small projects with them, but I'll be honest, I use my main Pi just for Kodi. After Codacademy I wanted to progress my learning so joined Treehouse for a couple of years, which I couldn't recommend enough! I found out how to use PHP properly and also about frameworks such as Flask, Django and Laravel.

I've started this site as a showcase of the small projects that I have completed / am completing. I've made a couple of scripts and sites to help with my day job. A lot of people ask why I began to learn programming, hopefully I can highlight where it has helped me with this site.
